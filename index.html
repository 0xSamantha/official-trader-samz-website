    Learning Resources

    hi!


    If you're looking to get into web3 development and don't know where to start here are some useful resources that have been personally helpful to me throughout my journey!


    Web3 Development

    'Gamified' Learning

    (best kind of learning?)
    • CryptoZombies - awesome for Solidity and a few other select chains
    • sol.pg - Go-to for Solana
    • freeCodeCamp Web3 - seperate section of fCC made specifically for web3!
    • OpenZeppelin's Ethernet - game for learning Web3/Solidity, requires metamask to play

    General Web3 Resources

    (for when you want to go more in-depth (locked in))
    • Solana Website and their YouTube Channel - in-depth Solana courses and guides, straight from the source. love how much time they put into educating, genuinely.
    • Alchemy University - comprehensive Solidity and general programming learning,
    • Ethereum Developer Guides - straight from the source
    • Open Zeppelin Learn - guide on smart contracts

    Regular Programming Resources

    (because we've all been there, seriously)
    • freeCodeCamp - my personal favorite, tons of free courses!
    • The Odin Project - great for full-stack development, learning the technical and real-world aspects of working as a developer
    • LeetCode - perfect for practicing your problem-solving skills
    • Codecademy - interactive learning for various programming languages
